# Python Introduction Exercises

This folder contains a small set of beginner-friendly Python examples and exercises used to introduce core language concepts.

## Structure

- `001_HelloWorld/`

  - `hello.py` — simple program that prints "Hello, World!".

- `002_Variables/`

  - `variables.py` — shows how to declare and use variables, basic types and printing.

- `003_BuiltinFunctions/`

  - `builtin.py` — examples of common built-in functions (e.g., `len`, `print`, `type`).

- `004_IfStatement/`

  - `if.py` — conditional statements and branching examples.

- `005_ForLoops/`

  - `loops.py` — `for` loop examples and iterating over lists/ranges.

- `006_WhileLoops/`

  - `while.py` — `while` loop examples and loop control.

- `007_CastingAndArithmeticalOperations/`

  - `casting.py` — type casting and arithmetic operations examples.

- `008_StringComparison/`

  - `stringcomp.py` — comparing strings and common string operations.

- `009_Match/`
  - `match.py` — `match`/`case` (structural pattern matching) examples (Python 3.10+).

## Quick start

Requirements

- Python 3.8+ (some examples like `match` require Python 3.10+).

Run an example (PowerShell)

From the `Introduction` directory you can run any example like this:

```powershell
# Run the Hello World example
python .\001_HelloWorld\hello.py

# Run variables example
python .\002_Variables\variables.py
```

## Tips

- If you have a virtual environment, activate it first.
- Use `python --version` to check your Python version. The `match` example requires 3.10 or newer.
- You can open these files in an editor and edit them to experiment interactively.
